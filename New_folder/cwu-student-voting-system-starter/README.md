# Cyprus West University Student Voting System

A professional frontend starter for a final year graduation project.

## Built With
- HTML
- CSS
- JavaScript
- LocalStorage demo data
- Chart.js CDN for results chart

## Pages
- index.html — landing page
- login.html — login screen
- dashboard.html — student dashboard
- elections.html — active/upcoming/closed elections
- vote.html — vote casting page
- results.html — election results
- admin.html — admin dashboard prototype

## How to Run
Open `index.html` in your browser or use VS Code Live Server.
